Expand description

Translates a point in the grid to a point on the screen.

A result of (0, 0) represents the top left corner of the screen. +x is right, +y is down. The function should never return a point farther than (canvas_width * (GRID_HEIGHT + 1), canvas_height * (GRID_WIDTH + 1)).
